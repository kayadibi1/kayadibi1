Johns Hopkins SAIS. Building reliability tooling for AI systems. Everything below ships with deterministic verification built in, not bolted on.

### What I'm working on now

[**fieldatlas**](https://github.com/kayadibi1/fieldatlas): automated literature review for AI safety where every claim is verified against source text and every citation is linted against a real corpus. Ground-truth tested on 15 papers blind to the system: **15/15 faithful, 0 fabrications, 0 material errors.** Generalized to a second field (CRISPR governance) on the first run.

[**nla-eval**](https://github.com/kayadibi1/nla-eval): interpretability evaluation of released NLA checkpoints on H100. Published negative results honestly. [Zenodo DOI](https://doi.org/10.5281/zenodo.20112029).
