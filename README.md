# Sidar Aslanoğlu

Johns Hopkins SAIS MA (International Relations: Security, Strategy and Statecraft). I build reliability and evaluation tooling for AI systems, plus OSINT and data pipelines. My target lane is the overlap of AI safety / Trust and Safety and political-risk / OSINT: GenAI-harms, adversarial-AI, and influence-operations analysis.

The throughline across the work below: deterministic guardrails and verification, with the limitations written into the code rather than hidden.

### Selected work
- **[fieldatlas](https://github.com/kayadibi1/fieldatlas)** — reliability-first automated literature review for AI safety and AI policy. Every read claim is verbatim-verified against source, every citation is linted against a real corpus, and the pipeline red-teams itself over six adversarial rounds. Tests pass; two fields run end to end.
- **[nla-eval](https://github.com/kayadibi1/nla-eval)** — an interpretability evaluation of released Natural Language Autoencoder checkpoints (Qwen2.5-7B, Gemma-3-12B) on an H100, with an anisotropy-corrected metric. Archived with a Zenodo DOI.
- **[multiverse-worldcup](https://github.com/kayadibi1/multiverse-worldcup)** — a browser-side Monte-Carlo World Cup oracle using IBM Granite and Docling (IBM SkillsBuild entry).
- **[emersusai](https://github.com/kayadibi1/emersusai)** — an evidence-grounded RAG health assistant over roughly 1.1M biomedical papers, with a refuse-rather-than-hallucinate grounding gate (archived portfolio artifact).
